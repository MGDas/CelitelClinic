3. Сделать первую миграцию.( python3 manage.py migrate )
4. Закомментировать все связные ключи с приложением doctor у: article, promotion, new, service
5. Закоментировать всё в файле admin у: article, promotion, new, service
6. Запустить скрипт run_migrate
7. Раскомментировать всё из пункта 4
8. Сново запустить скрипт run_migrate
